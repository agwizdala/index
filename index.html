Evolution of HTML

  • Tim Berners-Lee first made HTML in 1989. He did this to help scientists easily share their work with each other.
  • HTML 1.0 was the first verison of HTML, it was used from 1989 to 1994. This outdated coding made every website look the same because they couldnt add JPEG's or alter the backgrounds.
  • In 1995 2.0 had a bigger update where they could changde the background and use textg boxes and tables
  • Some web browsers had trouble showing pages the same way, which made things confusing for people.
  • W3C set rules for how HTML should work
  • in 1997 3.2 added cool features like tables and flowing texts around pictures
  • HTML 4.01, which came out in 1999, introduced "css" to make it easier to change how web pages look.
  • With CSS, the style of the webpage got separated from its main content, making web design much simpler.
  • HTML 5 is the newest version and supports high-def videos, fancy graphics, and works even when you're not online.
  • HTML keeps improving and now it's easier for many different web browsers to use its newest features.
